HTML5

Introducción

HTML5 o lenguaje de marcas de hipertexto en su versión 5, es un lenguaje de marcado para el desarrollo de sitios web, el cual es regulado por W3C, su lanzamiento inicial fue 28 de octubre del 2014, el cual cuenta con nuevos elementos y atributos para el uso de sitios web modernos.

Elementos Semánticos

Header es el elemento que siempre se utiliza para marcar la parte de la cabecera de una página. El cual contiene el logotipo de sitio web y puede estar anidado con otras secciones.

Nav es el elemento que se utiliza para representar una parte de una página y que también puede estar enlazada a otras partes dentro de la página o a otras páginas.

Section es el elemento que se utiliza para representar a un documento, una sección de aplicación, un capitulo o que contenga un encabezado propio.

Footer es el elemento que se utiliza para el pie de la página o sección, que contiene información como el autor, enlaces, información o derechos de autor, entre otros.

Aside es el elemento que se utiliza para representar una nota explicación o consejo.

Article es el elemento que se utiliza para representar una entrada de un blog, periódico o revista, entre otros.
