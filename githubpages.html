GITHUB PAGES

Introducción

GitHub Pages es un servicio de alojamiento que nos permite crear sitios web estáticos, y que trabaja los archivos en formato HTML, CSS y JavaScript, desde algún repositorio en GitHub, también permite la opción de ejecutar archivos a por medio de un proceso de compilación y la publicación de un sitio web.

Tipos de sitios de páginas de GitHub

Existen tres tipos de sitios de páginas de GitHub, los cuales son: proyecto, usuario y organización.

Los sitios de proyectos se encuentran conectados a un proyecto específico alojado en GitHub.

Los sitios de usuarios y organizaciones se encuentran conectados a una cuenta específica de GitHub.
