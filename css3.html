CSS3

Introducción

CSS3 o hojas de estilo en cascada en su versión 3, el cual es un lenguaje de diseño gráfico, ligado al lenguaje de marcado y construir el diseño visual del sitio web escritas en HTML5, y se utiliza con la función de crear paginas visualmente llamativas.

La sintaxis es sencilla y consiste en una serie de reglas, la cual consisten en uno o más selectores, además de un bloque de declaración.

Los selectores se aplican a elementos de un tipo y de un atributo en específico como el id que es un identificador de modo único de la etiqueta y del class o clase como identificar para varios elementos.

El uso de CCS3 se aplica para establecer los colores y el tamaño de las fuentes, el estilo de fondo, la alineación de elementos, bordes, entre otros.

Las ventajas de CSS son la accesibilidad, formateo de página, consistencia del sitio, ancho de banda, separación del contenido y la presentación.
