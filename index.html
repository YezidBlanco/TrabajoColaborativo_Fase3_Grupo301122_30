GIT/GITHUB, HTML5 Y CSS3

Descripción de la Temática General

Introducción, profundización e información complementaria sobre el uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web, además HTML5 y CSS3 para creación de contenido en formato web como páginas y sitios web.

Objetivo Específico 1

Realizar la instalación y configuración de los sistemas de control de versiones GIT y GITHUB.

Objetivo Específico 2

Crear un repositorio remoto y local, en los sistemas de control de versiones GIT y GITHUB.

Objetivo Específico 3

Desarrollar un sitio web OVI con el lenguaje de marcado HTML5 y con el lenguaje de hoja de estilos en cascada CSS3.
